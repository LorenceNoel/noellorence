JAVASCRIPT Cheat Sheet

JAVASCRIPT

JavaScript is a programming language that enables web developers to make websites interactive and dynamic. It allows you to add functionality to web pages, such as responding to user actions (like clicks and keystrokes), manipulating HTML content, and communicating with servers to load or send data without reloading the page. In simpler terms, JavaScript is what makes websites do cool stuff like animations, forms that validate input, and updating content without refreshing the entire page.

Variables

Use variables to store and manipulate data. Remember to use let, const, or var to declare variables, depending on the scope and mutability requirements.

let name = "John";

Functions

Functions encapsulate reusable blocks of code. They help improve code organization and maintainability by promoting modularity.

function greet() { console.log("Hello!"); }

onditional Statements

Use if, else if, and else to control the flow of your program based on different conditions. Conditions should evaluate to either true or false.

let age = 20; if (age >= 18) { console.log("You are an adult."); } else { console.log("You are a minor."); }

Loops

Loops help you execute code repeatedly. Use for, while, or do-while loops depending on the situation and termination conditions.

for (let i = 0; i < 5; i++) { console.log("Count:", i); }

Arrays

Arrays are collections of elements. You can access, modify, and iterate over array elements using various array methods and loop constructs.

let numbers = [1, 2, 3, 4, 5];

Objects

Objects store data in key-value pairs. They're useful for representing complex data structures and modeling real-world entities.

let person = { name: "Alice", age: 30, };

Event Handling

Use event listeners to respond to user interactions like clicks, keypresses, or form submissions. Event handling enables interactive web experiences.

document.querySelector("button").addEventListener("click", function() { console.log("Button clicked!"); });

DOM Manipulation

Modify HTML elements and their attributes using JavaScript to create dynamic and interactive web pages.

document.getElementById("demo").innerHTML = "Hello, World!";

Alerts

Display simple pop-up messages to provide information or prompt the user for action. Use alerts sparingly to avoid disrupting user experience.

alert("This is an alert!");

Prompt

Get input from users by displaying a dialog box with a text field. Prompt is useful for gathering user input in interactive applications.

let name = prompt("What is your name?");

Confirm

Display a confirmation dialog box with OK and Cancel buttons. Use confirm dialogs to prompt users for decisions with binary outcomes.

let confirmation = confirm("Are you sure?");

Timeouts

Execute code after a specified delay. Timeouts are useful for scheduling tasks to occur in the future, like animations or asynchronous operations.

setTimeout(function() { console.log("Delayed message"); }, 2000); // 2000 milliseconds = 2 seconds

Intervals

Execute code repeatedly at fixed intervals. Use intervals for tasks that need to be performed periodically, such as updating live data or running background tasks.

setInterval(function() { console.log("Interval message"); }, 1000); // 1000 milliseconds = 1 second

Error Handling

Use try, catch, and finally blocks to handle exceptions and gracefully recover from errors in your code. Error handling improves the robustness of your applications.

try { // Code that might throw an error } catch(error) { console.log("An error occurred:", error.message); }

AJAX

Use AJAX to make asynchronous HTTP requests to servers. Fetch data from APIs or send data to servers without reloading the entire web page, providing a seamless user experience.

fetch("https://api.example.com/data") .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error("Error fetching data:", error));
