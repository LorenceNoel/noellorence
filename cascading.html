CSS Cheat Sheet

CSS

Cascading Style Sheets. It's a style sheet language used for describing the presentation of a document written in markup languages like HTML. CSS describes how elements should be displayed on a screen, in print, or in other media. It allows web developers to control the layout, colors, fonts, and other visual aspects of web pages, making them more visually appealing and user-friendly.

CSS Selectors

This cheat sheet provides a quick reference for CSS selectors, which are patterns used to select and style elements in HTML documents. It includes examples of commonly used selectors such as tag selectors <p>, class selectors <.my-class>, and ID selectors <#my-id>, along with their corresponding CSS rules.

CSS Properties

Lists essential CSS properties that developers frequently use to style HTML elements. It includes properties like color, font-size, background-image, padding, margin, etc. Each property is accompanied by its syntax and a brief description of its purpose.

CSS Units

Provides a reference for different units of measurement used in CSS, such as pixels <px>, percentages <%>, em <em>, rem <rem>, and more. It helps developers understand how to specify sizes and distances in their CSS stylesheets.

CSS Flexbox

Focuses on CSS Flexbox layout properties, which are used for creating flexible and responsive layouts. It includes properties like display: flex, flex-direction, justify-content, align-items, etc. Each property is explained with examples to illustrate their usage.

CSS Grid

Covers CSS Grid layout properties, which enable developers to create complex grid-based layouts in a more structured way. It includes properties like display: grid, grid-template-columns, grid-template-rows, grid-column, grid-row, etc. Examples demonstrate how to define grid containers and place items within them.

CSS Animations

Focuses on CSS animation properties and keyframes, which allow developers to create animations directly in CSS without using JavaScript or external libraries. It includes examples of defining keyframes @keyframes, animation properties animation-name, animation-duration, animation-timing-function, etc., and applying animations to elements.

CSS Colors

Provides a reference for specifying colors in CSS using various formats such as hexadecimal #RRGGBB, RGB (rgb()), RGBA (rgba()), color names, and HSL (hsl()). It also includes examples of color manipulation functions like lighten(), darken(), saturate(), etc.

CSS Transitions

Covers CSS transition properties, which enable smooth transitions between different property values e.g., color, size, position over a specified duration. It includes properties like transition-property, transition-duration, transition-timing-function, and transition-delay, along with examples of their usage.

CSS Pseudo-classes

Explains CSS pseudo-classes, which are used to define special states of HTML elements that are not represented by the document structure. It includes examples of commonly used pseudo-classes like :hover, :active, :focus, :nth-child(), etc., along with their syntax and usage.

CSS Pseudo-elements

Focuses on CSS pseudo-elements, which allow developers to style specific parts of an element's content without adding extra HTML markup. It includes examples of pseudo-elements like ::before, ::after, ::first-line, ::first-letter, etc., along with practical use cases.

CSS Resets

Introduces CSS resets, which are stylesheets used to reset the default styles applied by browsers to HTML elements. It includes popular resets like Eric Meyer's Reset, Normalize.css, and others, along with their implementation.

CSS Media Queries

Covers CSS media queries, which allow developers to apply different styles based on the characteristics of the device or viewport. It includes examples of media query syntax for common use cases like targeting screen sizes, orientation landscape/portrait, and device features resolution, aspect ratio, etc..

CSS Typography

Focuses on CSS properties related to typography, such as font-family, font-size, font-weight, line-height, text-align, text-decoration, etc. It includes examples of styling text elements to achieve desired typography effects.

CSS Box Model

Explains the CSS box model, which describes how elements are rendered on a web page in terms of content area, padding, border, and margin. It includes examples of setting padding, margin, border properties, and how they affect the layout of elements.

CSS Specificity

Clarifies CSS specificity, which determines which CSS rule is applied to an element when multiple rules conflict. It includes examples and explanations of specificity rules based on selector types inline styles, IDs, classes, elements and how to calculate specificity values.
