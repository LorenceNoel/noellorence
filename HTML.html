HTML

Creating Custom Data Attributes

Use custom data attributes to store additional information about HTML elements. html Copy code <div> id="product" data-product-id="12345" data-category="electronics"></div>

Using HTML5 Canvas for Dynamic Graphics

Utilize the <canvas> element to draw graphics, animations, or interactive content dynamically. <canvas id="myCanvas" width="200" height="100"></canvas>

Implementing Web Components

Define reusable custom HTML elements with encapsulated styles and functionality. <my-custom-element></my-custom-element>

Embed interactive maps from services like Google Maps using <iframe> tags.

<iframe src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d3022..."></iframe>

Use the <template> element to define table row templates and generate dynamic tables with JavaScript.

<table>
  <tbody id="tableBody">
     <template id="rowTemplate">
       <tr>
            <td></td>
            <td></td>
      </tr>
    </template>
  </tbody>
</table>

Implementing Drag and Drop File Upload

Allow users to drag and drop files directly into the browser for uploading. <div id="dropZone" ondrop="dropHandler(event)" ondragover="dragOverHandler(event)"></div>

Implementing HTML5 Geolocation

Utilize HTML5 Geolocation API to retrieve the user's geographical location. <button onclick="getLocation()">Get Location</button>

Embedding Audio and Video with <audio> and <video>

Embed audio and video files directly into web pages with native HTML5 elements.
<audio controls>
   <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls>
   <source src="video.mp4" type="video/mp4">
</video>

Creating Responsive Images with srcset

Serve different image files based on the device's screen size and resolution. <img src="small.jpg" srcset="medium.jpg 1000w, large.jpg 2000w" alt="Responsive Image">

Using <details> and <summary> for Expandable Content

Description: Create collapsible sections of content that can be expanded or collapsed by the user.

<details>
    <summary>Click to expand</summary>
    <p>Hidden content here...</p>
</details>

Implementing Microdata with Schema.org

Add semantic markup to HTML to provide additional context for search engines.

<div itemscope itemtype="http://schema.org/Person">
   <span itemprop="name">John Doe</span>
</div>

Creating Progress Bars with <progress>

Display the progress of a task or process with the <progress> element.

<progress value="70" max="100"></progress>

Implementing Native HTML5 Drag and Drop

Use native HTML5 drag and drop API to enable dragging and dropping of elements within a web page.

<div draggable="true" ondragstart="dragStart(event)">Drag me!</div>
<div ondrop="drop(event)" ondragover="allowDrop(event)">Drop here!</div>

Creating Tabbed Navigation with <details> and <summary>

Implement tabbed navigation using <details> and <summary> elements.

<details open>
   <summary>Tab 1</summary>
   <p>Tab 1 content here...</p>
</details>

Implementing Accessibility with ARIA Roles and Attributes

Improve accessibility by adding ARIA roles and attributes to HTML elements. <button role="button" aria-label="Close">X</button>
